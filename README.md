- react-redux
- react-router-dom
- tailwindcss
- query-string
- axios

##  Installation
1. Clone the repository: (https://github.com/Anvo8222/web_blog_api_fe.git)
2. Install dependencies: yarn install
3. Yarn start
4. Open your web browser and visit http://localhost:3000
